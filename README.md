## Project Summary
This project implements a Rain Prediction App using [Python].  
The main features include:

- Data preprocessing and cleaning of weather datasets  
- Feature engineering and analysis  
- Building and training a predictive model for rainfall
- Interactive interface to predict rain based on input parameters  

The app demonstrates practical applications of machine learning in meteorology and decision support.

## Installation and Usage
1. Clone the repository:<br>
   git clone https://github.com/kecdevilla/PSMBE102_Rain_Prediction.git <br>
   cd PSMBE102_Rain_Prediction
2. Install dependencies: <br>
   pip install -r requirements.txt
3. Run the app <br>
   streamlit run app-final.py
4. You can also run the app via this link. Make sure to set Python version to 3.10: <br>
   https://psmdsbe102-rainapp.streamlit.app/ <br>
